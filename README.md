
# info

Do you want information about your packages?

## Authors

- [@ArticOff](https://www.github.com/ArticOff)


## Installation

Install info with pip

```bash
# Linux/macOS
python3 -m pip install -U infopkg

# Windows
py -3 -m pip install -U infopkg
```
    
## Usage/Examples

1. Open a terminal

2. Write this 
```bash
python -m info <YOUR PACKAGE NAME>
```

3. That's all !
